## What's Working

1. **Hero composition** carries the brand at the entry point — single warm shaft of light, Playfair clip-path word-reveal, Italian incantation in three words ("Forbice, mano, pazienza."), German clarification underneath. Image + accent + typography do the warmth work, never the bg.
2. **Opening-hours hierarchy** answers the user's question directly. Playfair italic clamp(2.25rem, 4vw, 3.25rem) carries the live message ("Öffnet Montag um 09:00 Uhr"); the seven-day strip below is supportive data, not the main act.
3. **Reviews editorial spread** breaks the services-card-grid reflex. Featured pull-quote with brass/10 Quote glyph behind it + two secondary marginalia notes with subtle "G Google" cues. Three motion tiers (single Reveal, 0.1s stagger) instead of three equal-weight cards.

## Priority Issues — Round 2

All addressed in this branch.

- **[P1] Lightbox + mobile nav lack focus trap** (a11y) → useFocusTrap hook applied to both modals; Tab + Shift+Tab now wrap to first/last focusable inside the dialog, restore on close
- **[P1] BookingButton ~32px tall** (responsive, WCAG 2.5.5 advisory) → min-h-11 + py-3 brings the primary CTA to 44px
- **[P1] Reviews section without data** → guard returns null when featured is undefined; static fallback still ships three reviews, so this is the edge-case belt
- **[P2] Cookie reset button is bare text** (responsive touch target) → min-h-11 + py-2 + focus-visible ring
- **[P3] Em-dashes in three aria-labels** (Logo, Footer social, Lightbox dialog) → replaced with commas, screen readers no longer announce "em dash Vorschau"
